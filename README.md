# Sales Forecasting Dashboard

A Sales Forecasting Dashboard built using **Python, Machine Learning, SARIMA, Streamlit, and Plotly**. The project performs data preprocessing, exploratory data analysis (EDA), sales forecasting, anomaly detection, and interactive dashboard visualization.

---

## Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Sales Forecasting using SARIMA
- Anomaly Detection using Isolation Forest
- Customer/Product Segmentation using K-Means
- Interactive Streamlit Dashboard
- Data Visualization with Plotly & Matplotlib

---

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/SalesForecasting_Vinal.git
```

Move into the project folder:

```bash
cd SalesForecasting_Vinal
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Run the Project

Run Data Preprocessing:

```bash
python file1.py
```

Run Forecasting & Machine Learning:

```bash
python file2.py
```

Launch the Streamlit Dashboard:

```bash
streamlit run app.py
```
